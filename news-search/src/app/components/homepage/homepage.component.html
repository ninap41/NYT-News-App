About

Basic Search platform using New York Times Api. Using Bootstrap 4, Angular Materials, Angular Routing Animations, NYTapi, Typescript, and Typescript classes to create dynamic queries for your news needs. ^_^

How it works

Select from the dropdown a news outlet (the only one available is NYT ¯\_(ツ)_/¯). The search bar should appear. Select a Page and type a key term/query. "News Desk" are topics and not neccesary to get results. If you want a response that's a pool from all News_Desk categories, just search a query.
