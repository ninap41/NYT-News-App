About

About: Basic Search platform using New York Times Api. Using Bootstrap 4, Angular Materials, Angular Routing Animations, NYTapi, Typescript, and Typescript classes to create dynamic queries for your news needs. ^_^

How it works

Select from the dropdown a news outlet (the only one available is NYT ¯\_(ツ)_/¯). The search bar should appear. Select a Page and type a key term/query. "News Desk" is like a topics category, but is not neccesary if you want general articles from al categories from your query.
